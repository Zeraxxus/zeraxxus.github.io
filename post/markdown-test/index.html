Code block

C:\_ singlelinecode.exe

void FancyAlert(char* word) {
  if(word) {
    printf("%s is fancy!", word);
  }
}

code is displayed using three grave accents ` or tildes ~ before and after the code, like this:

```
void FancyAlert(char* word) {
  if(word) {
    printf("%s is fancy!", word);
  }
}
```

or

~~~
void FancyAlert(char* word) {
  if(word) {
    printf("%s is fancy!", word);
  }
}
~~~
